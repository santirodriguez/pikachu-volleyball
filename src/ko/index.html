원조 피카츄 배구 (対戦ぴかちゅ～　ﾋﾞｰﾁﾊﾞﾚｰ編) 게임은 다음 분들이 만들었습니다.

이 프로젝트는 원작을 바탕으로 한 JavaScript 리버스 엔지니어링/재구현 작업을 기반으로 합니다.
작업 공개: 이규태

이 포크의 소스 코드: GitHub에서 볼 수 있습니다.

업데이트 기록

이 포크는 원조 게임과 이규태님의 JavaScript 재구현 작업을 이어서 개발됩니다.

Linux AppImage 지원과 품질 개선을 계속 진행하면서 웹 버전도 유지합니다.

조작법:

블루투스 키보드를 연결하면 스마트폰이나 태블릿에서도 플레이할 수 있습니다.

팁:

1) 1인 플레이("혼자서재미있게~")에서 Z 키를 누르면 1P, Enter 키를 누르면 2P로 시작합니다.

2) 1P는 아래 입력으로 V 키를 추가로 사용할 수 있습니다. V 키는 F 키와 동작이 다릅니다.

3) Esc 키로 메뉴 바를 숨기거나 다시 표시할 수 있습니다.

4) 연습 모드는 "설정 → 연습모드 → 켜기"에서 활성화할 수 있습니다.

5) 오프라인으로 플레이하려면 브라우저 메뉴에서 앱으로 설치하거나 모바일에서 홈 화면에 추가하세요.

파워 히트 버튼을 누르면 바로 다시 경기할 수 있습니다

승리점수를 점으로 바꿀 수 없습니다. 이미 이 점수가 됐거나 넘어버렸습니다.

연습모드에서 승리점수를 바꾸는 건 의미가 없습니다.
연습모드를 끄려면 "설정 → 연습모드 → 끄기"를 눌러주세요.
